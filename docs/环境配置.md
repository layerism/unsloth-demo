# 环境配置

## 第一步：如何登录我们的 GPU 服务机器
例如，同学A分配的账号为 s001，分配的隧道端口为 27000，老师会群里提供 ip，那么登录命令为
```shell
ssh -L 27000:localhost:27000 s001@ip
```

## 第二步：激活训练环境
直接运行
```shell
source /home/ubuntu/env.sh
```

## 第三步：下载项目代码
把项目代码 git 到自己电脑上
```shell
git clone git@github.com:layerism/unsloth-demo.git
```
备注：如果老师更新了代码，请进入 unsloth-demo 运行 `git pull` 更新便可

## 第四步：同步项目代码到服务器
流程：把我的代码拉到你的电脑上，然后 vscode 开启 SFTP 同步到你的 GPU 服务器账户 remotePath 下
如何设置 SFTP 的远程编辑的配置：
1. 需要 vscode 安装 SFTP 插件，作者：Natizyskunk
2. 在项目目录下，创建一个 .vscode 的文件夹
3. 里面创建一个 sftp.json 配置，更改 username，remotePath，配置如下
4. 然后可以配置保存时候，自动同步代码便可
```json
[
    {
        "name": "4090-8x",
        "host": "12.12.12.12",
        "protocol": "sftp",
        "port": 22,
        "username": "s001",
        "remotePath": "/home/2023/s001/llm/",
        "password": "xxxx",
        "uploadOnSave": true,
        "downloadOnOpen": false,
        "useTempFile": false,
        "openSsh": false,
        "watcher": {
            "files": "**/*",
            "autoUpload": true,
            "autoDelete": true
        },
        "ignore": [
            ".vscode",
            ".DS_Store",
            ".ipynb_checkpoints",
            "__pycache__",
            "*cache"
        ]
    }
]
```
